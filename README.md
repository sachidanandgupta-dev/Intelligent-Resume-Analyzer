# Intelligent Resume Analyzer :-



## Overview

A Python application that automates resume screening by extracting candidate details and matching them against job requirements.



## Features

- Resume parsing

- Skill matching algorithm

- Match score calculation

- Hiring recommendation

- JSON report generation



## Project Structure

main.py

parser.py

matcher.py

report_generator.py

sample_resume.txt

requirements.txt



## Run Project



python main.py



## Output

analysis_report.json



## Match Score Formula



Match Score = (Matched Skills / Required Skills) × 100
